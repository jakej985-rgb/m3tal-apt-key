Package Installation

Configure the official package repository and install M3TAL packages securely on your Linux machine.

🚀 Quick Automatic Setup

Run the official setup pipeline to import keys, declare the stable repository source, sync packages, and download the system CLI tools automatically.

curl -fsSL https://jakej985-rgb.github.io/m3tal-apt-key/install.sh | sudo bash

What this script does:

  1. Downloads and de-armors the public GPG key into /usr/share/keyrings/m3tal-archive-keyring.gpg.
  2. Appends the repository line safely into /etc/apt/sources.list.d/m3tal.list.
  3. Performs a package update index synchronisation and triggers apt install m3tal.

🛡️ GPG Key Verification

Verify the public signature key signature manually to prevent MITM tampering or unauthorized packages:

# Download public key
curl -fsSL https://jakej985-rgb.github.io/m3tal-apt-key/public.key -o m3tal.key

# Verify GPG fingerprint
gpg --show-keys m3tal.key
